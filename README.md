# Homebrew Tap for Claude Workbench

Homebrew formulae for [claude-workbench](https://github.com/eqms/claude-workbench) — Rust-based TUI multiplexer for AI-assisted development.

## Installation

```bash
brew install eqms/claude-workbench/claude-workbench
```

Or add the tap first:

```bash
brew tap eqms/claude-workbench
brew install claude-workbench
```

## Updating

```bash
brew upgrade claude-workbench
```

## Supported Platforms

| Platform | Architecture |
|----------|-------------|
| macOS | Apple Silicon (arm64) |
| macOS | Intel (amd64) |
| Linux | amd64 |
| Linux | arm64 |
